# Data Folder for UVDHDS_transcriptomics

This folder contains all processed count matrices, sample metadata, IPA outputs, and pathway information relevant to the *UVDHDS_transcriptomics* project.  
Files in this directory provide the basis for all analyses presented in the repository and support full reproducibility and transparency for publication and re-use.

---

## Folder Contents

- **UV-DHDS_count_matrix_EKB_20240924_lexogen_20241016.tsv**  
  Count matrix of gene-level (mRNA/lncRNA) expression values for experiment/run 1 (processed with Kangaroo/Lexogen). Genes as rows, samples as columns.

- **UV-DHDS_count_matrix_EKB_20250410_lexogen_20250428.tsv**  
  Count matrix of gene-level (mRNA/lncRNA) expression values for experiment/run 2 (processed with Kangaroo/Lexogen). Genes as rows, samples as columns.

- **UV-DHDS_mirna_counts_mirtop.tsv**  
  miRNA expression count table generated by mirtop (after nf-core/smRNAseq pipeline). miRNAs as rows, samples as columns.

- **metadata_mRNA.csv**  
  Comprehensive sample metadata for mRNA/lncRNA experiments, including run, donor, cell type, passage, seeding, sequencing QC, and other relevant fields. Matches samples in count matrices.

- **metadata_miRNA.csv**  
  Corresponding sample metadata for miRNA sequencing experiments, detailing sample-level variables.

- **processed_data_mRNA.rds**  
  R object containing normalized, filtered, and annotated mRNA/lncRNA dataset as output by script `01_UV-DHDS_data_processing.R`.

- **processed_data_miRNA.rds**  
  R object containing normalized, filtered, and annotated miRNA dataset as output by script `04_UV-DHDS_processing_miRNA.R`.

- **marker_genes.csv**  
  Marker genes for different skin cell entities curated for this project and used for identity validation.

- **KEGG_TGF_BETA_SIGNALING_PATHWAY.v2025.1.Hs**  
  KEGG pathway gene set file for human TGF-beta signaling, used for targeted/pathway-specific enrichment analysis of transcriptional data.

- **REACTOME_SIGNALING_BY_TGF_BETA_RECEPTOR_COMPLEX.v2025.1.Hs**  
  Reactome pathway gene set file for human TGF-beta receptor complex signaling, used as above.

- **IPA_canonical_pathways.xlsx**  
  Ingenuity Pathway Analysis (IPA, Qiagen) results table containing scores, statistics, and pathway assignments for canonical signaling pathways.

- **IPA_upstream_regulators.xlsx**  
  IPA results table with predicted upstream regulators and associated statistical metrics.

---

## Notes

- All count matrices use official gene symbols or miRNA names as row names; see associated metadata for column/sample matching.
- Metadata files are provided in CSV for accessibility and link samples to raw/processed data.
- The `.rds` files allow for rapid re-loading/replication of the objects used in downstream R analyses as described in the `/R` folder.
- KEGG/Reactome gene set files may be included in downstream analyses or Supplementary Material depending on manuscript decisions regarding pathway focus.
- IPA output files are directly exported from the IPA GUI and used in scripts in `/R/IPA.R` for further analysis/figure generation.
